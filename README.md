# Kevin's Game Collection

Six browser-based games, vibe coded with Qwen 3.6 27B.

| Game | Type | Dir |
|------|------|-----|
| Super Mario Bros | Platformer | `mariobros/` |
| Tetris | Puzzle | `tetris/` |
| Silent Thrill | 3D Horror | `silenthrill/` |
| Theme Park | Tycoon | `themepark/` |
| Whack-a-Mole | Arcade | `whackamole/` |
| Zork | Text Adventure | `zork/` |

Open `index.html` in your browser to browse, or open any game's `index.html` directly to play.
